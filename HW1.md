# HW1 สรุปเนื้อหา
### เนื้อหา
ชื่อของฟังก์ชันบอกถึงเจตนาที่ต้องการเน้นย้ำ, การปรับโครงสร้างใหม่ เป็นการทำเพื่อปรับให้โปรเเกรมดูชัดเจนมากขึ้นโดยไม่ได้เปลี่ยนวัตถุประสงค์เดิม เเละไม่ทำให้สิ่งที่เราทำมาไม่สามารถใช้ต่อได้, การเปลี่ยนชื่อเป็นการเปลี่ยนให้เข้ากับสิ่งที่เราทำ เพื่อไม่ทำให้สับสน, การทำ Replace Temp with Query คือเปลี่ยนตัวเเปรชั่วคราวเป็นฟังก์ชัน,   Inline Variable ไม่ควรเรียกฟังก์ชันซ้ำหลายๆรอบ เพราะจะทำให้ทำงานช้าลง , Spilt Loop เเยกออกการทำงานซ้ำๆเเบบลูปออกมา
