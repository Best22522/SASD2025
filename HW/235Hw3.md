a) Utility Class

ตัวอย่าง: System.IO.Path

คลาสนี้ประกอบด้วย static methods เท่านั้น และไม่มีสถานะข้อมูลที่สามารถเปลี่ยนแปลงได้ ซึ่งเป็นลักษณะของ utility class ที่ช่วยทำงานซ้ำ ๆ ในหลายจุดของโปรแกรม

b) Mutable Class

ตัวอย่าง: System.Text.StringBuilder

ค่าของ object สามารถเปลี่ยนแปลงได้หลังจากสร้างแล้ว (เช่น .Append(), .Insert(), .Remove()) ซึ่งต่างจาก immutable type ที่ค่าภายในไม่เปลี่ยนหลังสร้าง

c) Immutable Class

ตัวอย่าง: System.String

ทุกครั้งที่มีการ “เปลี่ยนแปลง” string จะ คืนค่าสตริงใหม่ แทนที่จะเปลี่ยน object เดิม ซึ่งเป็นนิยามของ immutability (ค่านี้ไม่เปลี่ยนตลอดอายุของ instance นั้น)
